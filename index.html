Privacy-first design

Built with strong encryption and responsible data practices.

MindCare uses public/private key cryptography for messages. Public keys are stored server-side, while private keys are encrypted with a separate user-defined encryption password that is never stored on the server.

This privacy-focused model helps protect communication and aligns with school trust requirements.

✔ Messages are stored in encrypted form

✔ Distinct encryption password from account password

✔ Access controlled by role and school context
