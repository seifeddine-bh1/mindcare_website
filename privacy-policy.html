Privacy Policy for MindCare

Last updated: [Insert Date]

1. Introduction

Welcome to MindCare (“we”, “our”, or “us”).

MindCare is a mobile application designed to facilitate communication between students and school psychologists, provide mental well-being support, and offer tools such as mood tracking and task management.

This Privacy Policy explains how we collect, use, store, and protect your information.

By using MindCare, you agree to this Privacy Policy.

2. Scope of Use

MindCare is intended for students aged 13 to 18 within educational institutions.

Access to the app is managed by schools.

Parental or guardian consent is the responsibility of the school, not MindCare.

3. Information We Collect

3.1 Account Information

When using MindCare, we collect:

  • First name
  • Last name
  • Email address
  • Birthdate
  • School code
  • Profile picture (optional)

3.2 Communication Data

  • Messages exchanged between students and psychologists
  • Anonymous or identified conversations depending on user choice

🔐 Message Encryption

  • Messages are encrypted using a cryptographic system based on public/private keys.
  • Messages are stored in encrypted form.
  • Private keys are encrypted using the user’s password.
  • We do not store user passwords.

⚠️ Important clarification:

While messages are strongly protected, MindCare cannot guarantee absolute end-to-end encryption in the strict cryptographic sense. However, message content is designed to remain inaccessible without user credentials.

3.3 Mood and Well-being Data

  • Daily mood selections
  • Predefined tags (e.g., stress, anxiety, etc.)
  • Optional personal notes

3.4 Tasks and Activity Data

  • Tasks assigned by psychologists
  • Completion status

3.5 Technical Data

We may collect limited technical data such as:

  • Device type
  • Operating system
  • App usage logs (for debugging and performance)

4. How We Use Information

We use collected data to:

  • Provide and operate the MindCare service
  • Enable communication between students and psychologists
  • Support mental health tracking (mood monitoring)
  • Allow psychologists to assign and track tasks
  • Improve app performance and user experience
  • Ensure safety and prevent misuse

5. Anonymous and Identified Communication

MindCare allows two modes of communication:

🟢 Identified Mode

Psychologists can see the student’s identity (name, profile).

🟡 Anonymous Mode

The student’s identity is hidden. A random name is shown instead.

⚠️ However:

The system may still associate messages with a user internally for security and functionality purposes.

6. Data Storage and Hosting

  • Data is stored using Supabase.
  • Servers are located in France (European Union).

This means your data benefits from EU-level data protection standards (GDPR).

7. Data Sharing

We do not sell personal data.

We may share data only with:

  • Authorized school psychologists (within app functionality)
  • Technical service providers (e.g., Supabase)
  • Legal authorities if required by law
  • In cases necessary to protect user safety

8. Data Retention

We retain data only as long as necessary to:

  • Provide the service
  • Comply with legal obligations
  • Ensure system integrity

9. Account Deletion

Users can delete their account directly داخل التطبيق.

When an account is deleted:

  • Personal data is deleted or anonymized
  • Some data may be retained temporarily for legal or security reasons

10. Profile Visibility

Profile pictures and basic profile data may be visible داخل النظام.

There is no direct interaction between students.

11. Security Measures

We implement appropriate safeguards:

  • Encrypted data storage
  • Secure authentication mechanisms
  • Access control restrictions

However, no system can guarantee 100% security.

12. Children’s Privacy

MindCare is used by minors (13–18 years old).

  • Schools are responsible for obtaining parental consent
  • We minimize data collection to what is strictly necessary
  • We apply enhanced safeguards for student data
